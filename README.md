# 🌸 Yo Soy Yo - E-commerce

Aplicación web de comercio electrónico desarrollada como proyecto final.  
La plataforma permite visualizar productos, buscar diseños, gestionar un carrito de compras y administrar productos mediante un panel privado.

El proyecto fue desarrollado utilizando React y Firebase, aplicando buenas prácticas de componentes, diseño responsive y experiencia de usuario.

---

## 🚀 Tecnologías utilizadas

### Frontend
- React.js
- React Router DOM
- CSS Modules
- React Bootstrap
- React Icons
- React Toastify
- React Helmet

### Backend / Base de datos
- Firebase Firestore
- Firebase Authentication

### Almacenamiento de imágenes
- ImgBB API

### Herramientas
- Vite
- Git / GitHub
- Vercel

---

## ✨ Funcionalidades principales

### 🛍️ Usuarios
- Visualización de productos.
- Detalle individual de cada producto.
- Búsqueda de productos por nombre/diseño.
- Paginación del catálogo.
- Agregar productos al carrito.
- Visualización del carrito de compras.

### 🔐 Administración
- Inicio de sesión para administrador.
- Rutas protegidas.
- Gestión de productos:
  - Crear productos.
  - Editar productos.
  - Eliminar productos.
- Gestión de cupones.
- Actualización dinámica de productos.

---

## 📱 Diseño y experiencia de usuario

El proyecto fue desarrollado aplicando enfoque Mobile First:

- Diseño adaptable a dispositivos móviles, tablets y escritorio.
- Componentes responsive.
- Feedback visual mediante notificaciones.
- Uso de iconografía para mejorar la navegación.
